00042     in      YL, SREG            ;[-21]

00044 ;----------------------------------------------------------------------------
00045 ; Synchronize with sync pattern:
00046 ;----------------------------------------------------------------------------
00047 ;sync byte (D-) pattern LSb to MSb: 01010100 [1 = idle = J, 0 = K]

00049 ;The first part waits at most 1 bit long since we must be in sync pattern.
00050 ;YL is guarenteed to be < 0x80 because I flag is clear. When we jump to
00051 ;waitForJ, ensure that this prerequisite is met.

00081 ;we have 1 bit time for setup purposes, then sample again. Numbers in brackets
00082 ;are cycles from center of first sync (double K) bit after the instruction
